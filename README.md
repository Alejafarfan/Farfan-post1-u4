# Galería Interactiva - JavaScript

## Estudiante
Alejandra Farfan Duarte

## Descripción
Este proyecto consiste en una galería interactiva desarrollada con JavaScript puro que permite crear, eliminar y filtrar tarjetas dinámicamente utilizando manipulación del DOM y eventos.

## Funcionalidades
- Crear tarjetas con título, descripción y categoría
- Eliminar tarjetas
- Filtrar tarjetas por categoría
- Contador de tarjetas visibles
- Mensaje cuando no existen tarjetas

## Tecnologías utilizadas
- HTML5
- CSS3
- JavaScript ES6
- DOM API

## Cómo ejecutar el proyecto

1. Clonar el repositorio

2. Abrir la carpeta en Visual Studio Code

3. Ejecutar con **Live Server**

4. Abrir `index.html`
